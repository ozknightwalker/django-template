# Custom Django project template


## Features

- Django 3.0+
- Uses [Poetry](https://python-poetry.org/) for managing dependencies

## How to install

```bash
$ django-admin.py startproject \
  --template=https://github.com/ozknightwalker/django-template/archive/master.zip \
  --name=Procfile \
  --extension=py,md,env \
  project_name
```
